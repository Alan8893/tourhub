# TH-0068 — Catalogue CSV Import

Status: DONE

## Goal

Support safe mass loading of products and recipes from spreadsheet-compatible CSV files.

## Delivered

- product CSV import;
- recipe, component, and note CSV import;
- preview and apply API operations;
- transactional all-or-nothing behavior;
- duplicate and reference validation;
- row and field error reporting;
- frontend upload, paste, preview, apply, and template download workflow;
- comma and semicolon delimiter support;
- regression tests and quality-gate coverage.
