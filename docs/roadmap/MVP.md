# MVP Roadmap

## Goal

Create a complete single-user hiking preparation scenario.

## Completed

- Project creation;
- Meal plan generation foundation;
- Shopping list;
- Purchase checklist;
- Document export.

## In Progress

- Improve Project context;
- Persist MealPlan structure;
- Support realistic meal schedules;
- Support multiple dishes per meal.

## Success Criteria

Instructor can:

1. Create a hiking project.
2. Enter participants and duration.
3. Generate a meal plan.
4. Review shopping requirements.
5. Export preparation documents.
