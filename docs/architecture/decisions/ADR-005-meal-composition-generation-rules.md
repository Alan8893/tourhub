# ADR-005 — Meal Composition Generation Rules

Status:

Accepted

## Problem

A hiking meal is not a single dish.

Example:

Breakfast:
- porridge
- sandwiches
- tea

## Decision

For MVP, meal composition is generated by selecting multiple dishes for the same meal_type.

No additional Dish category model is introduced yet.

## Generation Rules

A meal may contain:

- main dish;
- additional dish;
- drink.

These roles are logical only and are not stored separately yet.

## Example

Day 1:

Breakfast:
- oatmeal
- sandwiches
- tea

Stored as multiple MealPlanItem records with:

same day_number
same meal_type

## Future

When menu editor appears, introduce explicit composition categories and user customization rules.
